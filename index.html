Belang van code

Code. Poeh. Dat klinkt ingewikkeld. Hackers die dag en nacht bezig zijn om systemen te kraken. Ja, dat is inderdaad code. Maar weet je wat ook code is? Een pak melk scannen bij de supermarkt. Of snel checken hoe laat je bus ook alweer aankomt. “Daar kan ik er nog wel honderd van opnoemen”, hoor ik je denken. Maar ja, sta je daar eigenlijk wel eens bij stil?

Coderen voor kinderen

De generatie die hier het meeste bij stilstaat, is de jonge generatie. Onze wereld is de afgelopen 20 jaar gedigitaliseerd en aangezien jongeren hiermee opgroeien, zijn zij de oudere generaties al relatief snel voorbij in hun digitale vaardigheden.

Naast dat code belangrijk is, is het ook heel leuk! Bereid jij de jeugd op een leuke manier voor op een digitaliserende toekomst?
